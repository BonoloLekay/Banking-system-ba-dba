## User Stories

- As a customer, I want to deposit money into my account so that I can increase my balance  

- As a customer, I want to withdraw money so that I can access my funds  

- As a customer, I want to view my account balance so that I can track my finances  

- As a bank staff member, I want to view customer accounts so that I can assist customers  

- As a system, I must record all transactions so that financial data is accurate
